			=======================================================
				Set Category Data Types ( Collections Data types)
			=======================================================
=>The purpose of SetCategory Data Types is that "To store Multiple Values either of Same Type or Different Type OR 
    Both the Types in single Object with Unique Values Only (Duplicates are not allowed)."
=>In Python Programming, we have 2 Data Types in Set Category. They are

					1. set (Both Mutable and Immutable)
					2. frozenset (Immutable)
