				===========================================================
					Reading the Data OR Input dynamically from Key Board
				===========================================================
=>To Read the Data OR Input dynamically from Key Board, we have Two Pre-Defined Functions and They are present 'builtins' module. They are

				1. input()
				2. input(Message)
------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. input()
------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax: varname=input()
=>Here input() is used for Reading any Type of Value from Key Board and placed in LHS Varname and whose type is <class,'str'>. we can convert str type value into Our Required Type by using Type Casting Functions.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. input(Message)
------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Here Message Represents "User Prompting Message".
=>Syntax:   varname= input(Message)
=>Here  input(Message) is used for Reading any Type of Value from Key Board and placed in LHS Varname along with User-Prompting Message and whose type is <class,'str'>. we can convert str type value into Our Required Type by using Type Casting Functions.
------------------------------------------------------------------------------------------------------------------------------------------------------------------
