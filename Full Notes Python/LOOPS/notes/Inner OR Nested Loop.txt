				==================================================
						Inner OR Nested Loop
				==================================================
=>The Process of Defining One loop Inside of another Loop is called Inner OR Nested Loop
=>The execution Process of Inner OR Nested Loop is that "For Every Value of Outer Loop, Inner loop Executed for Finite     Number of Times until Test Condition becomes False "
=>when we use break statement inside of any inner loop then PVM comes Out-off corresponding Inner Loop only But will not come totally out-off  all loops.
=>In Python Programming, we can use Inner Loops in 4 Ways. They are
------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach-1:   for loop in for loop
------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:     for varname1  in Iterable-Object1: # Outer Loop
			----------------------------------------
			----------------------------------------
			for varname2 in Iterable-Object2: # Inner Loop
				---------------------------------------
				---------------------------------------
			else:
				-------------------------------------
				-------------------------------------
		else:
			-------------------------------------------
			-------------------------------------------
------------------------------------------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------------------------------------------

				-----------------------------
				while(Test Cond2): # Inner loop

					-----------------------
				else:

					-------------------------

				------------------------------------
