			========================================================
					3. Relational Operators ( Comparision Operators)
			========================================================
=>The purpose of Relational Operators is that "To compare Two Values".
=>If two Values Connected with  Relational Operators  then It is called Relational Expression.
=>The Result of Relational Expression is Either True OR False.
=>The Relational Expression is also called Simple Test Condtion and whose result can be either True OR False.
=>In Python Programming, we have 6 Types of Relational Operators. They are
------------------------------------------------------------------------------------------------------------------------------------------------------------
SLNO		SYMBOL		MEANING			EXAMPLES
------------------------------------------------------------------------------------------------------------------------------------------------------------
1.				>		greater than			print(10>2)---------True
											print(30>40)--------False

2.				<		less than				print(10<20)-------True
											print(100<30)------False

3.				==		double equal to		print(10==10)-----True
					     (Equality Operator )		print(10==20)------False

4.				!=		not equal to			print(10!=10)-------False
											print(10!=20)-------True

5				>=		greater than			print(10>=10)------True
						or equal to			print(10>=20)------False

6.				<=		less than				print(10<=10)------True
						or equal to			print(10<=5)--------False	
------------------------------------------------------------------------------------------------------------------------------------------------------------
NOTE: when we use Raltional Operators with str data and special symbols then PVM considers their ASCII Values and later comparision takes place.
