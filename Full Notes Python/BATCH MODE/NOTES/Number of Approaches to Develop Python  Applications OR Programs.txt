Definition of Program
------------------------------------
=>Set of Optimised Instructions is called program for Peforming Certain Task OR Solve a Problem
=>Here Optimised Instructions are nothing Taking Less Time Complexity and Less Space Complexity.
------------------------------------
=>The purpose of Writing the Program is "To perform Certain Task OR Solve a Problem in Real Time".
=>When we any Program in Python Envrinment, We must save those Optimised Instructions some name with extension .py (FileName.py).
=>Hence Every Python Program must be save on FileName.py
-----------------------------------------------------------------------------------------------------------------------------------------------------------
=>In Python Programming  Envrinment,we have Two Approaches to Develop the Programs. They are

				1. Interactive Approach
				2. Batch Mode Approach
------------------------------------------------------------------------------------------------------------------------------------------------------
1. Interactive Approach
------------------------------------------------------------------------------------------------------------------------------------------------------
=>In  Interactive Approach, the python Programmer can Write One Statement at a time and Gets One Output at a time.
=>This Approach is useful to Test One Instruction at a time in the Program.
=>This Approach is Not Recommended to Use to develop Program for Solving Big Problems bcoz It is unable to save Program and not able to provide Re-Usability.

Examples Softwares
------------------------------
					1. Python Command Prompt
					2. Python IDLE Shell
-------------------------
Python Code
-------------------------
>>>a=10
>>>b=20
>>>c=a+b
>>>print(a)
>>>print(b)
>>>print(c)
------------------------------------------------------------------------------------------------------------------------------------------------------
2. Batch Mode Approach
------------------------------------------------------------------------------------------------------------------------------------------------------
=>In Batch Mode Approach, Python Programmer writes Group of Optimized Instructions and saved on Some File Name with extension .py (FileName.py)
=>Batch Mode Programmig allows the Programmer to save the Code and Provides Code Re-Usability.
