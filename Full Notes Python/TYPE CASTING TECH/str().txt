				===================================================
								5. str()
				===================================================
=>Here str() is used for Converting all types of Values into str type.
=>Syntax:   varname=str(int OR float OR bool or complex)
--------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=10
>>> print(a,type(a))----------------------10 <class 'int'>
>>> b=str(a)
>>> print(b,type(b))----------------------10 <class 'str'>
>>> b-----------------------------------------'10'
-----------------------------
>>> a=2.3
>>> print(a,type(a))--------------2.3 <class 'float'>
>>> b=str(a)
>>> print(b,type(b))--------------2.3 <class 'str'>
>>>b----------------------------------'2.3'
-----------------------------
>>> a=True
>>> print(a,type(a))-----------------True <class 'bool'>
>>> b=str(a)
>>> print(b,type(b))----------------True <class 'str'>
>>> b----------------------------------'True'
----------------------------------
>>> a=2+3.4j
>>> print(a,type(a))-----------------(2+3.4j) <class 'complex'>
>>> b=str(a)
>>> print(b,type(b))-----------------(2+3.4j) <class 'str'>
>>> b------------------------------------'(2+3.4j)'
-------------------------------------------------------------------------------------------------------------------------------------------------------
